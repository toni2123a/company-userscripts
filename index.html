Grundprogramm installieren

  1. Tampermonkey installieren
    Edge-Store öffnen
    Nach der Installation Edge ggf. neu starten.
  2. Entwicklermodus einschalten
    edge://extensions öffnen
    Den Schalter „Entwicklermodus“ findest du links mittig auf edge://extensions.
    Wenn der Link nicht öffnet, wird die Adresse automatisch in die Zwischenablage kopiert.
    Link kopiert. Öffne einen neuen Tab und füge edge://extensions ein.
  3. Tool auswählen & installieren
    Links ein Tool aufklappen und auf „Tool installieren“ klicken. Tampermonkey zeigt den Install-Dialog.
